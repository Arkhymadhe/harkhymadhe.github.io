Chemcaption

An open-source project worked on in collaboration with:
  • The Jablonka Group
  • The OpenBioML team

This project explores conversion of molecular fingerprints into text corpora. The aim of this conversion is to leverage existing cheminformatics databases to generate ready made cheminformatics text corpora. This would allow us leverage LLMs more effectively for chemistry.

As a major outcome of the project, over 40GB of data was generated for the Chemnlp project.
